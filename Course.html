Course Title:Web-development

Description:

This course introduces students to the fundamentals of web development. It focuses on building and designing websites using HTML, CSS, and basic JavaScript. Students will learn how web pages are structured, styled, and made interactive, along with best practices for creating responsive and user-friendly websites.

Syllabus:

  • Introduction to Web Development
  • How the Web Works (Basics of Internet & Browsers)
  • HTML Fundamentals (Structure, Tags, Forms)
  • CSS Basics (Colors, Fonts, Layouts)
  • CSS Box Model and Flexbox
  • Responsive Web Design Basics
  • Introduction to JavaScript, Basic DOM Manipulation
  • Final Project: Simple Website Development
